Maze Solver Program (Java)

GitHub
  • This program was a project for CSC111. It takes a text file that ideally contains x's and blank spaces and returns a path that successfully solves a maze. Otherwise, it will print that the maze is not possible.
  • I used Java to code this program; there were three methods!
  • I liked this project because it made me evolve my code to be more readable and useful by separating the code into their own methods, rather than piling everything into the main method.

Sorting Algorithms + Comparisons (Java)

GitHub
  • This program was another project for CSC111. It has three sorting algorithms: Selection, Bubble, and Merge.
  • Alongside with these methods, they have their corresponding method to calculate the amount of comparisons it would take to sort arrays of unsorted (and randomly generated) integers.


  • I liked this project because it made me think which step of the sorting process actually compares them.
  • Once I had understood where that step was, I was able to slightly tweak the original method to return the amount of comparisons.
